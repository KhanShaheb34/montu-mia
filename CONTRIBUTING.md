# Contributing to Montu Mia's System Design

Thank you for your interest in contributing to **Montu Mia's System Design**! We welcome contributions from everyone to help make system design concepts easier for Bengali speakers to understand.

## How Can You Contribute?

### 1. Reporting Bugs

- If you find any typo, incorrect information, or broken link, please open an issue.
- Clearly describe the problem and where it is located.

### 2. Suggesting Enhancements

- Have an idea for a new topic? Or a better analogy?
- Open an issue describing your suggestion.

### 3. Pull Requests

- Fork the repository.
- Create a new branch (`git checkout -b feature/AmazingFeature`).
- Commit your changes (`git commit -m 'Add some AmazingFeature'`).
- Push to the branch (`git push origin feature/AmazingFeature`).
- Open a Pull Request.

## Guidelines

- Please keep the tone **casual, fun, and easy to understand**, just like Montu Mia would explain.
- Use **Bengali** for the main content.
- Ensure all analogies are relatable to the local context.
